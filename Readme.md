# Symfony 5 & ReactJS boilerplate
 - Stack :
    - Traefik
    - PHP7.4 / nginx
    - Symfony 5
      - Monolog
    - ReactJS
    - React admin
    

What will you find here
- A back office to manage the front
- A PWA for the front office made with reactJS
- An api which is consumed both by the front and the backoffice
